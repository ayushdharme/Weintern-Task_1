# To-Do App

A simple and lightweight To-Do application built using **HTML, CSS, and JavaScript**.  
It helps users add, manage, and track daily tasks efficiently in a clean UI.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Dynamic task updates without page reload
- Simple and responsive design

## Tech Stack
- **HTML** – Structure
- **CSS** – Styling
- **JavaScript** – Logic and interactivity
